🚀 SUPER RÁPIDO (5 PASSOS)
════════════════════════════════════════════════════════════════════

PASSO 1: Obter Token do Bot
────────────────────────────
• Abrir Telegram
• Procurar @BotFather
• Enviar: /newbot
• Seguir instruções
• Copiar token (algo como 123456789:ABCDefGHijklmnoPQRstuvwxyz)

PASSO 2: Configurar .env
────────────────────────────
• Copiar .env.example para .env
• Abrir .env e preencher:
  TELEGRAM_BOT_TOKEN=seu_token_aqui
  API_URL=https://seu-painel.com/telegram_api.php
  ADMIN_IDS=seu_id_telegram (ex: 123456789)

Para descobrir seu ID: mandar mensagem no bot e ver ID nos logs

PASSO 3: Instalar e Testar Local
────────────────────────────
npm install
npm start

Você deve ver:
✅ Bot iniciado com sucesso!
🔗 Modo: Polling
📡 API URL: https://seu-painel.com/telegram_api.php
👮 Admins configurados: 123456789

PASSO 4: Testar no Telegram
────────────────────────────
• Abrir bot no Telegram (@seu-bot-username)
• Enviar /start
• Deve receber menu com opções
• Clicar em "📋 Servidores"
• Deve aparecer lista de servidores

PASSO 5: Deploy Gratuito (Railway - 30 min)
────────────────────────────
• Fazer push código para GitHub
• Abrir railway.app
• Conectar GitHub
• Railway faz deploy automático
• Pronto! Bot rodando 24/7 grátis

💾 COMO IMPORTAR SERVIDORES EM BULK
════════════════════════════════════════════════════════════════════

1. Criar arquivo servidores.txt:
   ───────────────────────────
   Servidor Brasil 1|http://url1.com
   Servidor Brasil 2|http://url2.com
   Servidor Premium|http://premium.com.br

2. No bot, enviar comando: /importar

3. Enviar o arquivo .txt

4. Bot importa automaticamente
   Resultado: ✅ Adicionados: 3

Pronto! Servidores cadastrados.


📡 INTEGRAÇÃO COM SEU PAINEL
════════════════════════════════════════════════════════════════════

Arquivo telegram_api.php DEVE estar:
  → No MESMO servidor que seu painel web
  → Na mesma pasta ou raiz do painel
  → Com acesso ao banco de dados (db.php)

Exemplo:
  Se painel está em: https://seu-painel.com/
  Copiar telegram_api.php para: https://seu-painel.com/telegram_api.php

Testar:
  curl "https://seu-painel.com/telegram_api.php?action=get_servers"
  
  Deve retornar JSON com servidores
