🚀 COMO COMEÇAR
═══════════════════════════════════════════════════════════════════

PASSO 1: Criar Token do Bot
────────────────────────────
• Abrir Telegram e conversar com @BotFather
• Comando: /newbot
• Escolher nome e username
• Copiar token fornecido

PASSO 2: Configurar .env
────────────────────────────
cp .env.example .env

Editar .env com:
  TELEGRAM_BOT_TOKEN=seu_token
  API_URL=https://seu-painel.com/telegram_api.php
  ADMIN_IDS=seu_id_telegram

PASSO 3: Instalar Dependências
────────────────────────────
npm install

PASSO 4: Testar Localmente
────────────────────────────
npm start

Deve ver:
  ✅ Bot iniciado com sucesso!

PASSO 5: Fazer Deploy Gratuito
────────────────────────────
Escolha uma opção:

🥇 RAILWAY.APP (Recomendado)
   - Mais fácil, não precisa cartão
   - 500 horas/mês grátis
   - Webhook automático
   Passos: GitHub → Railway Dashboard → Deploy automático

🥈 RENDER.COM
   - Alternativa ao Railway
   - Interface parecida
   - 750 horas/mês grátis

🥉 GOOGLE CLOUD RUN
   - Mais robusto
   - 2 milhões requisições/mês
   - Requer Google Cloud SDK

📌 VPS/SHARED HOSTING
   - Se tiver acesso SSH
   - Usar PM2 para manter rodando


⚡ INTEGRAÇÃO COM SEU PAINEL
═══════════════════════════════════════════════════════════════════

O arquivo telegram_api.php deve estar no MESMO SERVIDOR do painel.

1. Copiar telegram_api.php para raiz do painel
   scp telegram_api.php user@seu-servidor:/caminho/painel/

2. Verificar que usa mesmo db.php do painel
   ✅ Automático: require 'db.php'

3. Testar endpoint:
   curl "https://seu-painel.com/telegram_api.php?action=get_servers"

4. Se retornar JSON com "ok":true, está funcionando!


💾 ESTRUTURA DE BANCO DE DADOS
═══════════════════════════════════════════════════════════════════

O bot usa as tabelas existentes do seu painel:
  • servers (id, name, url, from_m3u)
  • m3u_accounts (id, username, password, host, ...)
  • telegram_users (opcional, criada auto)

Nenhuma migração necessária!

🐛 TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════════════════════

❌ Bot não responde:
   • Verificar token em .env
   • Testar: curl https://api.telegram.org/botTOKEN/getMe

❌ API retorna 404:
   • Confirmar URL em API_URL
   • Copiar telegram_api.php para servidor
   • Testar endpoint manualmente

❌ Erro de conexão:
   • Verificar internet
   • Confirmar credenciais do banco (db.php)
   • Ver logs: pm2 logs bot-iptv

❌ Variáveis não carregam:
   • Railway/Render: usar Dashboard, não .env
   • Local: restart do bot
   • Verificar syntax do .env

Ver SETUP_GUIA.md para troubleshooting completo.
